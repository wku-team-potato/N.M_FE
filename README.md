# N.M_FE
- Nutrition Manager 안드로이드 애플리케이션
- MVVM 패턴을 기반으로 한 안드로이드 애플리케이션

## 프로젝트 구조
```
N.M_FE
├── application
│   ├── common
│   ├── data
│   │   ├── model
│   │   ├── repository
│   │   ├── service
│   ├── ui
│   │   ├── view
│   │   ├── viewmodel
│   ├── utils
```
